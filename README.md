# Python GUI Alarm Clock

## Description
A fully functional desktop alarm clock built with Python and Tkinter. Features such as:
- Multiple alarms
- Snooze button
- Custom popup window with emoji
- Continuous alarm sound

## Technologies
- Python
- Tkinter
- Winsound (Windows)

## How to Run
1. Open terminal in project folder
2. Run: `python alarm_gui.py`
3. Enter time in HH:MM format
4. Click "Set Alarm" and wait until comes
